# 👻 Pac-Man (Java)

A simple **Pac-Man clone developed in Java using Swing**.  
The player controls Pac-Man through a maze, collecting pellets while avoiding ghosts.

# 🎮 Gameplay

* The player controls **Pac-Man** inside a maze.
* Pac-Man must **eat all the pellets** in the level.
* **Ghosts chase Pac-Man** through the maze.
* If a ghost touches Pac-Man, the **game ends**.

---

# ⌨️ Controls

| Key | Action |
|-----|------|
| ↑ Up Arrow | Move Up |
| ↓ Down Arrow | Move Down |
| ← Left Arrow | Move Left |
| → Right Arrow | Move Right |

---

# ⚙️ Technologies Used

* **Java**
* **Java Swing**
* **JPanel rendering with `paintComponent()`**
* **Keyboard input with `KeyListener`**
* **Game loop with `ActionListener`**
* **Tile-based map system**

# ▶️ Run the Game

### 1️⃣ Clone the repository

```
git clone git@github.com:NourBelghazi/PacMan.git
```

### 2️⃣ Compile

```
javac *.java
```

### 3️⃣ Run

```
java App
```

# 🧠 Concepts Used

This project demonstrates several important programming concepts:

* Object-Oriented Programming
* Event handling with `KeyListener`
* Game loop using `ActionListener`
* Rendering graphics with `paintComponent()`
* Collision detection
